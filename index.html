About me
I'm Péter Panyik, a year old undergraduate student, currently studying Computer Science Engineering, at John von Neumann University in Hungary.

Right now I'm a hobbyist programmer, who's mainly interested in gameplay programming (mostly C#), shader programming and vfx.

My current studies include: general programming in C# (using .Net), Java and C++; creating desktop and mobile applications using WPF, Xamarin and Xcode; low level programming in Assembly; databases, full stack web development, and a bit of game development.

I have a passion for programming and game development, I started programming at the age of 13, using Unity.

My motivation came from popular sandbox games, such as Garry's Mod, Minecraft, Terraria. I love to create things, and game development had a promise of this creative freedom.

Bedsides IT, I also know a bit of electronics, since I studied it as a subject in High School, and sometimes I build circuits for Arduino, along with a bit of C++ coding.
